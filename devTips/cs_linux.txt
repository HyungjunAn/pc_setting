tail: 파일의 뒷 부분 출력

-------------------------------------------------------------
# file: 파일의 종류 확인

-------------------------------------------------------------
# wild card
-------------------------------------------------------------
*				// 모든 문자
?				// 모든 하나의 문자
[characters]	// characters 문자셋에 포함된 문자
[!characters]	// characters 문자셋에 포함되지 않은 문자
[[:class:]]		// 지정된 문자 클래스에 포함된 문자
[:alnum:]		// 모든 알파벳과 숫자 문자
[:alpha:]		// 모든 알파벳 문자
[:digit:]		// 모든 숫자
[:lower:]		// 모든 소문자
[:upper:]		// 모든 대문자

-------------------------------------------------------------
# uniq: 중복된 행 삭제
-------------------------------------------------------------
$ uniq <FILE>
-c: 중복 행 개수 출력
-d: 중복 행 중 한 행만 출력
-D: 중복 행 모두 출력
-u: 중복 행 아닌 것만 출력

-------------------------------------------------------------
# sort: 파일 내용 정렬
-------------------------------------------------------------
$ sort <FILE>
-f: 대소문자 구별 안 함
-r: 역순
-o: 저장할 파일명 지정
